# APRS Messenger — Future Features

## SAR-specific
- [ ] Canned messages — configurable quick-reply buttons ("Copy", "In position", "RTB", "All clear")
- [ ] Position beacon — periodically send own position to APRS-IS
- [ ] Group broadcast — send same message to all saved stations with one click
- [ ] Message templates — pre-built messages with fill-in-the-blank fields

## Map / Situational Awareness
- [ ] Embedded map — show heard/saved stations on Leaflet/OpenStreetMap in web UI
- [ ] Distance/bearing — show how far each heard station is from your position

## Quality of Life
- [ ] Sound/desktop notification on new message
- [ ] Message search — search through message history
- [ ] Export log — download messages.log as CSV, filterable by date/station
- [ ] Dark/light theme toggle
- [ ] Mobile-friendly sidebar — slide-out menu instead of hidden on mobile
- [ ] Systemd service — run web UI as always-on service on the Z840

## Advanced
- [ ] Weather alerts — subscribe to NWS APRS weather bulletins
- [ ] Object/item posting — place markers on APRS map (staging areas, hazards)
- [ ] Bulletin board — read/post APRS bulletins and announcements
